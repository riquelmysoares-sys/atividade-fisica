O que é HTML?

HTML (HyperText Markup Language) é uma linguagem de marcação, utilizada para estruturar páginas da web.
Ela não é uma linguagem de programação, pois não possui lógica, variáveis ou estruturas condicionais.
O HTML serve para organizar e estruturar o conteúdo de um site (textos, imagens, links, etc.).


Estrutura básica de um documento HTML

Todo documento HTML deve começar com:

<!DOCTYPE html> → informa ao navegador que estamos usando HTML5

<html> → envolve todo o conteúdo da página

<head> → contém informações da página (título, configurações, etc.)

<body> → contém o conteúdo visível da página

Glossário das principais tags
<h1> até <h6>

São usadas para títulos e subtítulos.
<h1> é o mais importante e maior, <h6> é o menor.

<p>

Define um parágrafo de texto.

<a>

Cria um link para outra página.
Exemplo:

<a href="https://google.com">Google</a>
<img>

Insere uma imagem na página.
Exemplo:

<img src="imagem.jpg" alt="Descrição">
<div>

É uma tag usada para organizar e agrupar elementos.
